verde¶

The Verde public API.

Classes

verde.BaseGridder Base class for gridders.
verde.BlockMean(spacing[, region, adjust, …]) Apply a (weighted) mean to the data in blocks/windows.
verde.BlockReduce(reduction, spacing[, …]) Apply a reduction/aggregation operation to the data in blocks/windows.
verde.Chain(steps) Chain filtering operations to fit on each subsequent output.
verde.ScipyGridder([method, extra_args]) A scipy.interpolate based gridder for scalar Cartesian data.
verde.Spline([fudge, damping, shape, …]) Biharmonic spline interpolation using Green’s functions.
verde.Trend(degree) Fit a 2D polynomial trend to spatial data.
verde.Vector2D([poisson, fudge, damping, …]) Elastically coupled interpolation of 2-component vector data.
verde.VectorTrend(degree) Fit 2D polynomial trends to multi-component vector data.

verde.block_split(coordinates, spacing[, …]) Split a region into blocks and label points according to where they fall.
